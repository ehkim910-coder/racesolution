DRIVING MODES

현장 상황에 맞게,
3가지 방식

설비를 바꾸지 않아도 됩니다. 지금 현장 그대로, 바로 자동화를 시작할 수 있습니다.

Mode 01 · Follow-Me

추종 주행

라이다 센서로 작업자를 인식해
뒤를 따라다니며 짐을 나릅니다.
사람이 무거운 걸 들 필요가 없어집니다.

Mode 02 · Memory Trace

자동 주행

한 번 알려준 경로를 기억해
자동으로 주행합니다.
정해진 길을 하루 종일 대신 오갑니다.

Mode 03 · Line Trace

라인 주행

바닥에 반사 테이프만 붙이면
정해진 라인을 따라 움직입니다.
공사 없이 경로가 완성됩니다.

LINE-UP

3가지 모델
