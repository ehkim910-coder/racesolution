SOLUTION

복잡한 자동화는 그만
로봇 한 대가 현장에 맞춰 움직입니다

매핑도, 서버도, 별도 설비도 필요 없습니다.
THOUZER는 현장 상황에 맞춰
세 가지 방식으로 일합니다.

Mode 01 · Follow-Me

추종 주행

라이다 센서로 작업자를 인식해
뒤를 따라다니며 짐을 나릅니다.
사람이 무거운 걸 들 필요가 없어집니다.

Mode 02 · Memory Trace

자동 주행

한 번 알려준 경로를 기억해
자동으로 주행합니다.
정해진 길을 하루 종일 대신 오갑니다.

Mode 03 · Line Trace

라인 주행

바닥에 반사 테이프만 붙이면
정해진 라인을 따라 움직입니다.
공사 없이 경로가 완성됩니다.

외부 프로그램도, 설비 투자도 없이
로봇 본체 하나면 즉시 시작됩니다.
